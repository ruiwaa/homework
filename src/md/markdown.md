# 마크다운 문법

---

## 1.Markdown(마크다운이란?)

### 1) 마크다운의 시작

존 그루버는 2004년에 문법 면에서 에런 스워츠와 중대한 협업을 통해 마크다운 언어를 만들었으며 사람들이 읽기 쉽고 쓰기 쉬운 플레인 텍스트 포맷을 사용하여 쓸 수 있으면서 유효한 XHML(또는 HTNL)로 선택적 변환이 가능하게 하는 것을 목표로 이 언어를 개발하게 되었다.

### 2) Markdown의 특징

마크다운은 일반 텍스트 기반의 **경량 마크업 언어**다. 일반 텍스톨 서석이 있는 문서를 작성하는 데 사용되며, 일반 마크업 언어에 비해 문법이 쉽고 간단한 것이 특징이다. HTML과 리치 텍스트(RTF)등 서식 문서로 쉽게 변환되기 때문에 응용 소프트웨어와 함께 배포되는 **README파일**이나 온라인 게시물 등에 많이 사용된다. 특히, 마크다운 형식의 텍스트 파일은 github에서 버전 관리하는데 용이하므로 github README파일에서 자주 이용되어진다.

## 2. Markdown의 문법

#### 1) 문단 구분

문단과 문단 사이는 빈 줄(엔터키 두 번)로 구분한다.

#### 2) 문단 제목 추가

문단 제목은 "#제목" 형식을 이용하여 표현한다. (#1개부터 #6개까지 가능)

#### 3)서체

1)이탤릭체
이탤릭체는 문자를 입력한 후 문자 앞뒤로 '\*' 또는 '-'를 입력한다.

`예시`
_이탤릭체_ 입니다, _이탤릭체_ 입니다.

2)볼드체
볼드체는 문자를 입력한 후 문자 앞뒤로 '\*\*' 또는 '\_\_'를 입력한다.

`예시`
안녕하세요. 저는 **장예지**입니다. 벌써 **2026년** 이 다가오네요.

#### 4) 취소선

작성한 내용 중에 취소하고 싶은 내용은 해당 내용에 '~~'로 감싸준다.

`예시`
~~취소선~~은 ~~를 사용하면 됩니다.

#### 5) 하이퍼링크

하이퍼링크는 [링크명](해당 링크 주소 입력)
[링크명](http://www.example.com)

#### 6) 인용은 각 문단의 첫 줄 앞에 '>'를 붙인다.

#### 7) 리스트

##### -순서가 있는 리스트

순서가 있는 리스트를 만들기 위해서는 1.,2.,3. 숫자를 입력어를 추가하여 나타낸다. 순자순서대로 넣지 않아도 리스트 적용이 잘 부여된다.

`예시`

> 숫자 순서대로의 리스트
> 1.:smile:
> 2.:heart:
> 3.:fire:

> 같은 숫자가 반복되는 리스트
> 1.💓
> 1.❤️
> 1.🧡

##### -순서가 <u>없는</u> 리스트

순서가 없는 목록을 나타내려면 - 또는 \*를 한 번 사용하고 띄운다.

`예시`

> -오늘 할일
> *청소하기
> *16기 프론트엔드 과제하기
> \*1주차 학습한 내용 복습하기

#### 8) 텍스트 상자

문장이나 단어 앞 뒤에 `(백틱 기호)를 사용하여 텍스트 상자를 생성할 수 있다.

`예시`
`마크다운 문법을 활용한 텍스트 상자 만들기`

#### 9) 코드 블록

프로그래밍 코드를 마크다운을 활용해 입력 할 때는 코드의 앞과 뒤에 백틱 기호 3개 또는 ~~~를 사용하여 포현한다. 앞에 오는 3개의 백틱 또는 틸트 다음에 어떤 프로그래밍 언어를 작성한 것인지 적으면 언어의 문법에 맞춰 하이라이트 표시가 된다.

`예시`

```ts
console.log('Hello World!');
```

#### 10) 이미지 첨부

이미지를 삽입하려면 !+[링크명]+() 순으로 작성한다. 대괄호 안에는 대체 텍스트, 소관호 안에는 이미지의 URL를 작성한다.

`예시`
![무한도전 웃긴 이미지](https://i.pinimg.com/736x/b0/b1/a6/b0b1a693d5a136f29f675b08e6ea123b.jpg)
![무한도전 웃긴 이미지 2탄](https://i.pinimg.com/736x/43/60/55/43605577efee9a1ef28f80bffbd3f23b.jpg)

#### 11) 표 작성

---과 |를 이용해서 표를 만들 수 있다. 행을 구분할 때 ---를 사용하고 열을 구분할 때는 |을 사용한다. 셀의 내용을 정렬할 때는 :를 사용한다.

`예시`
베스킨라빈스 선호하는 맛 조사
|성명|선호하는 맛종류|
|:-----:|:-----:|
|장예지|초코나무 숲|
|A|엄마는 외계인|
|B|레인보우 샤베트|
